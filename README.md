## 📌 Assignment Overview

This assignment covers fundamental concepts of **Functions** and **Recursion** in C language, including:
1. Power function (`a^b`).
2. Sum of digits of a 5-digit number (Iterative + Recursive).
3. Fibonacci Sequence generation (user-defined limit).
4. Output prediction of recursive programs.
